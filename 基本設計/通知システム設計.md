
## 1. 通知システム設計

### 通知インターフェース
```python
class NotificationService(ABC):
    @abstractmethod
    async def send_approaching_notification(self, reception: Reception):
        """順番接近通知"""
        pass
    
    @abstractmethod
    async def send_call_notification(self, reception: Reception):
        """呼び出し通知"""
        pass
```
